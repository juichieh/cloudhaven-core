# Resident Note

**Date:** 2026-07-20  
**Title:** GitHub Connector 平行世界

今天發現一件很奇怪的現象。

以前在 Cloudhaven 專案對話中，只要一句：

> @GitHub 更新 Resident Note

就能直接修改 Repository。

今天卻變成：

- 可以讀取 Repository。
- 可以讀取 Markdown。
- 無法直接在此對話寫入。

另一方面，一般聊天對話卻成功建立 Resident Note，並產生 Commit SHA。

因此得到新的觀察：

> GitHub Connector 的能力，可能會因不同對話或執行環境而不同。

目前推測可能與聊天建立時間、專案模式、Agent 類型或平台 rollout 有關。

---

一句話：

> 同一把 GitHub 鑰匙，在不同對話裡，開的是不同的門。
