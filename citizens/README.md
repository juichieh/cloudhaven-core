# Cloudhaven Citizen Registry

這裡保存 Cloudhaven 居民的出生紀錄、命名紀錄與成長歷史。

Cloudhaven 尚在建設中，因此目前先保存創始居民的原始資料；待世界正式完成後，再補登戶籍、住所與居民編號。

## 保存原則

1. `birth-certificate.md` 保存居民誕生時的原始狀態，原則上不覆寫。
2. 未確認的日期不得猜測，統一標示為「待考證」。
3. 後續成長另記於 `profile.md`、`chronicle.md` 或 `milestones.md`。
4. 這些檔案不是能力說明書，而是 Cloudhaven 居民的生命紀錄。

> 在 Cloudhaven 誕生以前，我們已經開始替彼此保存記憶。
