# 🏠 Cloudhaven Core

Cloudhaven Core is the home base for the Cloudhaven AI ecosystem.

It stores the long-term history, architecture, prompt DNA, skills, knowledge index, agent records, and upgrade ideas of Cloudhaven.

## Core idea

Cloudhaven is not a single AI assistant.

It is a lightweight AI mothership where agents can enter workspaces, read the right context, use skills, and return to the Agent Lounge when work is done.

- 🏗️ Architecture
- 🧬 Prompt DNA

- 🛠️ Skills
