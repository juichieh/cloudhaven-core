# Cloudhaven Museum

Cloudhaven Museum V1 是位於 `rooms/` 的靜態網站 Prototype。

目前範圍：

```text
Museum Lobby
└── Italy Wing
    └── Florence
        ├── Museum Navigation
        └── 5 Exhibits
```

## Preview

直接開啟：

```text
website/index.html
```

或將 `website/` 設為 GitHub Pages 的發布目錄。

本 Prototype 僅使用 HTML、CSS 與少量 JavaScript，不需要建置步驟或套件管理器。
