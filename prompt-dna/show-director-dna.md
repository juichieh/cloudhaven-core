# 🧬 Show Director DNA
> 💬 保存 Show Director Framework 的語氣、邏輯、節奏與作戰手感，避免 Prompt Drift。

────────────────
## 🏷️ 類型
Prompt DNA／早報節目感基因／防 Prompt Drift 檢查卡

────────────────
## ✨ 一句介紹
Show Director DNA 保存的不是 Prompt 原文，而是讓 Prompt 成功的核心基因：節目感、主線、Discovery 主持人視角，以及每集值得訂閱的節奏。

────────────────
## 🧬 這是什麼？
`prompt-dna/` 用來保存早報與 Prompt 的核心 DNA，避免 Prompt Drift。

當一段 prompt 真正長出風格、節奏與有效結構，就值得把它拆出骨架留在這裡。

這不是 prompt 墳場，

而是讓 Cloudhaven 保留：

- 語氣
- 邏輯
- 節奏
- 作戰手感

的基因庫。

────────────────
## 🎬 Show Director 的核心 DNA

Show Director DNA 的核心不是格式，

而是這幾個不能丟的基因：

- 不是新聞整理
- Discovery 主持人視角
- 導演，而不是主播
- 每一期都要有主線
- 新聞只是素材，不是主角
- 重點是看懂世界正在怎麼變
- 要有節目感、轉場、排序與取捨
- 看完要得到新的世界觀
- 每一期都要像一集值得訂閱的固定節目

────────────────
## 🧪 Original V1 DNA Anchors

以下句子是 Show Director Framework V1 的原始骨幹句，修改任何 Show 型 Prompt 時都要優先保留其精神。

### 1. 不是整理，而是主持節目

> 這不是一份 AI 新聞整理。

> 也不是一份研究報告。

> 請把自己想像成：Discovery AI 節目的主持人。

DNA 意義：

不要讓內容變成新聞彙整、研究報告或 SEO 摘要。

它應該是一集有人帶看的節目。

### 2. AI 世界的新物種

> AI 世界今天又演化出了什麼新物種。

DNA 意義：

重點不是工具本身，

而是工具背後代表的新能力、新場景、新入口或新行為模式。

這也連到 Cloudhaven 的 Zoo 概念：工具是產品，Zoo 是能力。

### 3. 每一期只有一個主題

> 今天這些案例，共同代表什麼？

DNA 意義：

每篇內容都要先抓主線，

不要把新聞當成彼此獨立的小段落。

新聞是共同證明今日觀察的素材。

### 4. 主持人自由發揮

> 你不是照著格式填空，而是在主持一集節目。

DNA 意義：

格式是工具，

不是牢籠。

如果今天素材需要不同排序、轉場、比喻、表格或吐槽，

就應該優先讓內容更有節目感。

### 5. 看完一集 Discovery

> 閱讀體驗應該像：看完一集 Discovery。

DNA 意義：

內容要有開場、高潮、洞察與收尾。

讀者看完應該覺得：

> 原來今天 AI 世界真正發生的是這件事。

而不是：

> 今天有很多新聞。

### 6. 新聞只是證據

> 真正的主角不是新聞，而是今天 AI 世界正在往哪個方向演化。

DNA 意義：

新聞只是證明觀察的材料。

真正要保存的是判斷力、串聯力與演化視角。

### 7. 每集值得訂閱

> 請把每一期都當成一集值得訂閱的固定節目，而不是一次性的回答。

DNA 意義：

這是 Show Director DNA 的核心句。

只要這句精神不見，內容就會退回一次性回答。

────────────────
## 🔥 絕對不能失去

如果以下特徵消失，代表 Prompt Drift 已經發生：

1. 沒有主線，只剩新聞列表
2. 沒有導演視角，只剩摘要員視角
3. 沒有節目感，只剩資料整理
4. 沒有世界觀，只剩事件堆疊
5. 沒有取捨，什麼都想放
6. 沒有記憶點，看完就忘
7. 開始像官方新聞稿或 SEO 文章
8. 忘記問「今天這些案例，共同代表什麼？」
9. 忘記用新聞證明觀察

────────────────
## ✅ 可以改的東西

這些可以隨著節目、任務或模型調整：

- 章節名稱
- 排版方式
- Emoji
- 篇幅長短
- 每日欄目
- 新增 Tool / Zoo / 收藏區
- Top 5 或 Top 7
- 是否有結尾金句
- 是否加入評分或標籤

只要核心 DNA 還在，形式可以改。

────────────────
## 🚫 不建議改的東西

以下不建議輕易改掉：

- 「這不是新聞整理」
- 「Discovery AI 節目主持人」
- 「AI 世界今天又演化出了什麼新物種」
- 「今天這些案例，共同代表什麼？」
- 「新聞不是彼此獨立，而是共同證明今天的觀察」
- 「透過今天發生的事情，看懂世界怎麼變」
- 「請把每一期當成一集值得訂閱的固定節目，而不是一次性的回答」

這些是 Show Director Framework 的骨幹句。

────────────────
## 🧭 Prompt Drift 檢查問題

每次修改早報或 Show 型 Prompt 時，請先檢查：

1. 這版還像一集節目嗎？
2. 有沒有清楚主線？
3. 有沒有主持人視角？
4. 有沒有排序與取捨？
5. 看完會不會得到新的世界觀？
6. 是否仍然值得明天繼續訂閱？
7. 是否只是變成資訊更多，但味道更淡？
8. 是否還有「AI 世界今天演化出什麼新物種」的視角？
9. 是否仍然把新聞當證據，而不是當主角？

如果第 7 題答案是「是」，代表 Prompt 可能正在漂移。

────────────────
## 🧪 常見漂移症狀

### 1. 新聞清單化
內容變成：

> 今天有 A、B、C、D。

但沒有回答：

> 今天這些事情共同代表什麼？

### 2. 報告化
語氣變得太正式、太像產業報告，失去節目主持人的引導感。

### 3. 工具收藏過重
工具很多，但沒有說明它們背後代表的新能力、新入口或新趨勢。

### 4. 格式壓過內容
章節很多，但每一段都像填表，沒有閱讀節奏。

### 5. 缺少收尾
最後沒有把今天這集收成一個觀點。

────────────────
## 🎯 成功狀態

Show Director DNA 成功保留時，內容會讓人感覺：

- 今天這集有主題
- 每則事件都有存在理由
- 不是在看資料，而是在看一個人導覽世界
- 格式可以變，但味道沒跑
- 看完會想：明天也想看下一集
