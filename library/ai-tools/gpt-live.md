# 🆕 GPT-Live

> 🎙️ 即時語音模型／對話介面

────────────────

## 🏷️ 類型

即時語音模型／對話介面

────────────────

## ✨ 一句介紹

能同時聽與說，讓插話、打斷與語音輪替更接近真人對話。

────────────────

## 👍 推薦原因

- 對桌面小夥伴與生活型 Agent 很重要。
- 比單純提高音質更接近真正互動。
- 適合追蹤未來鉗鉗語音介面的發展方向。

────────────────

## ⏳ 時效性

🟡 中：正在逐步推出與持續更新。

────────────────

## ⭐ 收藏等級

⭐⭐⭐ 必收

────────────────

## 🧬 為什麼它算卡片

GPT-Live 雖然是模型／介面名稱，但它代表的不是單純「聲音比較好聽」，而是一個新的 Agent 互動能力：

- 同時聽與說
- 支援自然插話與打斷
- 更接近真人的語音輪替
- 讓 Agent 從語音輸入工具，進化成可持續互動的生活夥伴

因此它既是一張具體 AI 工具卡，也可以作為未來 Zoo 能力卡「Realtime Conversational Agent」的對應工具來源。

────────────────

## 🎯 觀察重點

- 打斷與插話是否真的自然？
- 多人環境下能否分辨誰在說話？
- 延遲是否足以支撐桌面小夥伴與生活型 Agent？
- 能否和記憶、視覺、工具使用串成完整互動？
- 是否能穩定處理長時間對話與情緒變化？

────────────────
