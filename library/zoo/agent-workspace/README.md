# 🏢 Agent Workspace／Agent 工作空間

> 觀察 Agent、工作流、MCP、知識庫、人工審批與企業服務如何被整合成同一個工作環境。

────────────────

## 🏷️ 分類定位

Agent Workspace / Open-source Agent Platform / Agent Operating Environment

這類卡片主要觀察：

- Agent 是否從單體工具變成可協作的工作台
- Agent、MCP、知識庫與自動化流程是否能共同運作
- 狀態、權限、審批、重試與錯誤處理如何被整合
- 是否開始出現真正的「Agent 居住環境」
