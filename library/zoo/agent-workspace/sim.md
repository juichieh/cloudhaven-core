# 🆕 Sim

> 🏢 Agent Workspace／開源 Agent 平台

────────────────

## ✨ 一句介紹

把 Agent、工作流、MCP、知識庫與企業服務放進同一個開源工作空間。

────────────────

## 🌱 成熟度

🌱 已可玩，仍快速成長

────────────────

## ⭐ 收藏價值

★★★★★

────────────────

## 👍 為什麼收藏

- 它不是單一 Agent，而是「Agent 居住環境」這個新物種。
- 和 Cloudhaven、鉗鉗的外部工作層高度相關。
- 它能用來觀察 Agent 工具是否從單體能力，演化成整合式協作工作台。
- 很適合對照 Cloudhaven 的 Room、Skill、Tool、Permission 與 Human Approval 分層。

────────────────

## 🎯 觀察重點

- Agent、工作流、MCP、知識庫與企業服務是否真正整合？
- 是否具備清楚的狀態管理、人工審批與權限邊界？
- 多 Agent 協作是實際能力，還是只是視覺化包裝？
- 錯誤處理、重試、可觀測性與長時間任務是否可靠？
- 能不能成為鉗鉗的外部 Workspace，而不取代 Lobster Core？

────────────────

## 🧬 Evolution Log

### 2026-07-12｜🆕 新發現

首次收錄為 Agent Workspace 類新物種。Sim 的重點不是「多一個 Agent 工具」，而是把 Agent、工作流、MCP、知識庫與企業服務聚合成共同工作的環境。

────────────────

## 🧬 AI Zoo 觀察

Sim 代表的物種是：

> Agent 不再只是一個會執行任務的個體，而是開始需要自己的工作台、工具牆、知識區、審批流程與協作環境。

這和 Cloudhaven 的概念高度相似，但兩者定位不同：

- Sim：外部 Agent 工作空間／執行平台
- Cloudhaven：長期知識、角色、技能、架構與工作室母艦

兩者未來可能是互補關係，而不是替代關係。
