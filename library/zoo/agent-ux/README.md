# 🌐 Agent UX／Agent 體驗與機器可操作介面

> 觀察網站、App 與服務如何不只服務人類，也能讓代表人類行動的 Agent 正確理解、操作與驗證結果。

────────────────

## 🏷️ 分類定位

Agent UX / Machine-operable Interface / Agent-ready Design

這類卡片主要觀察：

- 網站是否具備清楚、穩定、可理解的語意結構
- Agent 是否能可靠完成登入、搜尋、比較、填表、購買與驗證
- 操作結果是否可被檢查，而不是只靠畫面猜測
- 人類 UX 與 Agent UX 如何共存
- 是否逐步形成跨網站的 Agent-ready 標準

────────────────

## 🧭 適合收錄

- Agent-Ready Website
- Machine-readable UX
- Semantic Action Interface
- Verifiable Agent Interaction
- Agent-safe Checkout
- Browser Agent Design Framework

────────────────

## 🦞 龍蝦一句話

「網站以前只怕人類找不到按鈕；以後還要怕 Agent 看懂錯按鈕。」
