## 👍 為什麼收藏

它代表網站設計正在出現新的訪客類型：

> 不只服務人類，也要服務代表人類行動的 Agent。

這會直接影響未來：

- 旅行搜尋與訂購
- 電商與付款
- 表單填寫
- 跨網站比較
- Agent 驗證任務是否成功

────────────────

## 🎯 觀察重點

- 網站是否提供穩定、清楚的語意與操作結構？
- Agent 是否能確認價格、條件、權限與最終結果？
- 人類 UX 與 Agent UX 是否會互相衝突？
- 是否會形成通用標準、schema 或驗證協議？
- 如何防止 Agent 誤操作、暗黑模式與惡意誘導？
