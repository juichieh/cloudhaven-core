# 🧠 Memory／跨系統共享記憶

> 觀察 AI 記憶如何從單一聊天室附屬資料，演化成能跨 App、跨模型、跨裝置與跨身體移動的獨立層。

────────────────

## 🏷️ 分類定位

Shared Memory / Portable Memory / Cross-App Memory Layer

這類卡片主要觀察：

- 記憶能否跨 App 與模型共用
- 角色是否能帶著長期經驗遷移
- 同步衝突、隱私、刪除與權限如何治理
- 記憶是否可驗證、可編輯、可攜帶
- 記憶層與模型、Agent、身分層如何解耦

────────────────

## 🧭 適合收錄

- Shared Memory
- Portable Memory
- Cross-App Memory
- Memory Sync
- Character Memory Layer
- Memory Governance

────────────────
