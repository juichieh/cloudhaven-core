# 🦁 Keyboard Agent
> 💬 AI 住進鍵盤，從「開 App」變成「邊打字邊叫 AI」。

────────────────
## 🏷️ 能力類型
輸入法 Agent／Mobile Agent／Agentic Interface

────────────────
## ✨ 一句介紹
讓 AI 直接存在於鍵盤或輸入入口中，協助搜尋、整理、改寫與行動操作。

────────────────
## 🧠 這是什麼能力？
Keyboard Agent 指的是：

AI 不再只是獨立 App，

而是整合到使用者正在輸入文字的地方。

例如：

- 鍵盤
- 輸入法
- 搜尋列
- 訊息輸入框
- 系統文字框

────────────────
## 🤔 為什麼重要？
因為這代表 AI 入口正在從：

「我去打開 AI App」

變成：

「AI 出現在我正在做事的地方」。

這可能比語音助理更自然，也更適合手機使用。

────────────────

## 🧬 演化狀態
🌱 Seed / 萌芽

────────────────
## 📈 觀察重點
- 手機鍵盤是否會成為 AI Agent 入口
- 是否能理解上下文
- 是否能從輸入情境延伸出實用操作
- iOS / Android 是否會原生整合
- 是否會補足部分語音助理做不好的使用情境
