# 🌐 Context.dev

> 讓 AI 抓取、豐富化並理解網頁內容的一個 API。

────────────────

## 🏷️ 類型

Web Context API / Agent Web Layer

────────────────

## ✨ 一句介紹

讓 AI 抓取、豐富化並理解網頁內容的一個 API。

────────────────

## 🌱 成熟度

🌱 可研究

它代表 Agent 巡網、整理資料、抽取上下文時，可能需要一層專門把 Web 轉成乾淨資訊的基礎設施。

────────────────

## ⭐ 收藏價值

★★★★☆

────────────────

## 👍 為什麼收藏

- 鉗鉗未來巡網路、整理工具圖鑑，很需要這種把 web 變成乾淨上下文的能力。
- 它屬於 Agent Web Layer，而不是單純搜尋工具。
- 很適合觀察未來「Web → Context → Agent Action」這條資料流如何成熟。

────────────────

## 🎯 觀察重點

- 能否穩定抽取網頁主內容？
- 是否能保留來源、引用與上下文結構？
- 對工具圖鑑、早報、研究任務是否能減少雜訊？
- 是否能處理動態網頁、文件、產品頁、社群頁？
- 未來能不能成為鉗鉗巡網與 Cloudhaven 收藏流程的一層？

────────────────
