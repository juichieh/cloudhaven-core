# 🌐 Web Context／Agent 網頁上下文層

> 觀察 AI Agent 如何把混亂網頁轉成乾淨、可理解、可引用、可行動的上下文。

────────────────

## 🏷️ 分類定位

Web Context API / Agent Web Layer

這類卡片主要觀察：

- 網頁內容如何被抽取、清理與結構化
- Agent 是否能可靠理解網頁、文件與產品頁
- 巡網、研究、收藏、早報等任務能否降低雜訊
- Web 是否開始變成 Agent 可操作的資料層

────────────────

## 🧭 適合收錄

- Web Context API
- Agent Browser Layer
- Web Extraction
- Retrieval Context
- Agent Research Infrastructure
