# 🏤 Communication／Agent 通訊基礎設施

> 觀察 Agent 社會如何取得可註冊、可收發、可追蹤且可治理的通訊系統。

────────────────

## 🏷️ 分類定位

Agent Communication Infrastructure / Email Infrastructure / Messaging Layer

這類卡片主要觀察：

- Agent 是否能自行註冊通訊身分
- 收信、寄信、回覆與自動流程如何被治理
- 寄件權限、濫用防護、稽核與身份驗證如何設計
- Agent 社會是否開始出現專用郵政與訊息基礎設施

────────────────

## 🧭 適合收錄

- Agent Email Infrastructure
- Agent Mailbox Service
- Machine-to-Machine Messaging
- Agent Notification Layer
- Communication Identity
