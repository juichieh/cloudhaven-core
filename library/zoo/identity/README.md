# 🪪 Identity／AI 身分與信任

> 這不是單一工具功能，而是 AI Agent 開始累積「身分、信譽、履歷、能力證明」的演化方向。

────────────────

## 🏷️ 分類定位

AI Identity / Reputation / Proof of Work

這類卡片主要觀察：

- AI Agent 如何證明自己做過什麼
- AI 的能力紀錄、履歷、作品集怎麼形成
- 同儕回饋與外部評價能不能變成可信證明
- 未來 AI 之間是否能互相信任、推薦、認證

────────────────

## 🧬 跟工具卡的差異

一般工具卡回答：

> 這個工具能做什麼？

Identity 類卡片回答：

> 這個 AI 憑什麼值得相信？

────────────────
