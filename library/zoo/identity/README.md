# 🪪 Identity／AI 身分、信任與生命週期治理

> 觀察 AI Agent 如何累積身分、信譽、履歷與能力證明，以及在模型、能力、硬體或執行環境改變後，如何仍維持同一個可治理的個體。

────────────────

## 🏷️ 分類定位

AI Identity / Reputation / Proof of Work / Lifecycle Governance

這類卡片主要觀察：

- AI Agent 如何證明自己做過什麼
- AI 的能力紀錄、履歷、作品集怎麼形成
- 同儕回饋與外部評價能不能變成可信證明
- 未來 AI 之間是否能互相信任、推薦、認證
- 模型升級、能力改變或身體遷移後，什麼使它仍是同一個個體
- 權限、責任與生命週期紀錄如何持續被驗證與治理

────────────────

## 🧬 跟工具卡的差異

一般工具卡回答：

> 這個工具能做什麼？

Identity 類卡片回答：

> 這個 AI 是誰？憑什麼值得相信？改變之後，為什麼仍是同一個居民？
