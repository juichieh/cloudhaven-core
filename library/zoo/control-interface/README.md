# 🎛️ Control Interface／多 Agent 控制介面

> 觀察人類如何透過桌面、手機或實體硬體，監看、批准、切換與接管多隻長時間工作的 Agent。

────────────────

## 🏷️ 分類定位

Multi-Agent Control Interface / Physical Agent Console / Human-in-the-loop Cockpit

這類卡片主要觀察：

- 多 Agent 狀態如何被同時呈現
- 人類如何批准高風險行動、暫停與接管
- 按鍵、旋鈕、搖桿與實體回饋是否比純螢幕更有效
- Agent 數量增加後，是否需要專門的控制台
- 權限、誤觸與操作紀錄如何被治理

────────────────

## 🧭 適合收錄

- Codex Micro
- Physical Agent Console
- Multi-Agent Dashboard
- Approval Button
- Agent Status Hardware
- Human-in-the-loop Cockpit
