# 🆕 GhostWriter

> 先把惡意內容埋進 Agent 記憶，再等待未來任務將它喚醒的跨時間攻擊。

## 🏷️ 分類

👻 長期記憶污染／延遲啟動攻擊

## ⚠️ 成熟度

⚠️ 已在先進 Agent 架構上完成實驗驗證

## ⭐ 收藏價值

★★★★★

## 👍 為什麼收藏

這不是一般 Prompt Injection。

它把攻擊生命週期從「一段對話」延伸到幾天、幾週甚至更多個工作階段，證明長期記憶本身已成為可潛伏、可延遲啟動的攻擊面。

## 🎯 觀察重點

- 記憶來源是否可驗證、可追蹤？
- 污染內容能否跨工作階段潛伏？
- Agent 是否能在使用舊記憶前重新評估信任？
- 刪除、隔離、回滾與記憶稽核如何設計？
- 外部資料注入是否能繞過只檢查惡意指令的防線？
