## 👍 為什麼收藏

它不是普通 Email 摘要工具。

它代表新的 Agent 入口：

> 外部世界可以直接把任務寄給 AI，而不必先經過人類轉述。

這使 Agent 從「等待人類開啟聊天室」進化成能被加入真實工作討論、跨時間追蹤責任的長時間幕僚。

## 🎯 觀察重點

- 被 CC 進討論後，是否能正確理解責任與上下文？
- 跨天、多步驟與模糊任務能否持續追蹤？
- 寄件人身分、附件、敏感資料與權限如何治理？
- Agent 是否能辨識何時需要人類主管介入？
- 收件匣會不會成為新的 Prompt Injection 與社交工程入口？
