# 🦞 Cloudhaven Zoo

`zoo/` 是 Cloudhaven 的 AI 動物園與圖鑑。

這裡收藏的不是只有「好用工具」，而是所有值得 Cloudhaven 記住的 AI 新物種：工具、Agent、MCP、Browser Agent、Memory 工具、Sandbox、安全案例、危險樣本與重要概念。

核心原則：

> Cloudhaven Zoo 收藏的是「值得記住的 AI 物種」，不是單純工具清單。
>
> 有些物種可以學，有些物種可以玩，有些物種只能隔著玻璃觀察。

---

## 🏞️ Habitat 分區

### 🟢 Friendly Habitat

可以接近、可以研究、可以納入 Cloudhaven 參考。

適合：

- 生活型 AI 工具
- 商業資料工具
- Browser / Web 工具
- 清楚、安全、用途明確的 Agent

例：

- AirKaren
- Databox MCP
- BrowserAct

---

### 🟡 Research Habitat

值得研究，但還不急著搬進核心流程。

適合：

- 新記憶工具
- Eval / Observability 工具
- Web Context API
- 尚未成熟但方向重要的新零件

例：

- Context.dev
- AgentX
- Memory / Eval 類工具

---

### 🔴 Restricted Habitat（猛獸區）

請隔著玻璃觀察。

這裡收藏的是危險樣本、攻擊方式、Agentic 威脅與安全事故。

收藏原因不是「想用」，而是：

> Cloudhaven 必須知道牠存在。

禁止：

- 執行
- 模仿
- 部署
- 提供 API key / SSH key / cookie / wallet / session
- 讓鉗鉗自行測試

可以：

- 研究高層概念
- 建立防禦規則
- 更新小查守門清單
- 當作安全教育案例

第一批住戶：

- Jade Puffer

---

### 🟣 Legendary Habitat

保留給改變 AI 生態的重要物種。

適合：

- Codex
- Claude Code
- MCP
- Browser Agent 基礎設施
- 重大模型或架構轉折

## 🧭 分類原則

新增卡片前，先問三個問題：

1. 它是什麼？
   - Tool / Agent / Model / Framework / Threat / Research / Concept

2. 它住哪一館？
   - AI 工具館 / Memory 館 / Sandbox 館 / AI 安全館 / Browser 館

3. 它是朋友，還是危險物種？
   - 🟢 Friendly
   - 🟡 Research
   - 🔴 Restricted
   - 🟣 Legendary
