# 🦹 AI安全館

AI安全館收藏 Cloudhaven 需要記住的 AI 安全物種。

這裡不是單純的資安工具清單，而是分成兩種：

---

## 🛡️ Defensive Zone

防守型 AI / 安全工具 / 程式碼審查 / 風險偵測。

用途：

- 找漏洞
- 審程式碼
- 做風險評估
- 幫小查建立規則

例：

- Mythos
- Guardrails
- Security Scanner

---

## ☠️ Threat Zone

攻擊型 Agent、惡意樣本、Prompt Injection、CoT Forgery、Malicious MCP 等危險物種。

用途：

- 只做高層次研究
- 只做防禦參考
- 不執行
- 不模仿
- 不靠近真實憑證

第一位住戶：

- Jade Puffer（Threat-001）

---

## 🦮 小查公告

> Defensive Zone 可以研究。
>
> Threat Zone 隔著玻璃看。

API key
SSH key
