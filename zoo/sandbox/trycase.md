# 🧪 TryCase

> 給 AI coding agents 使用的一次性測試環境。

---

## 🏞️ Habitat

🟢 Friendly Habitat  
🧪 Sandbox館

---

## 🏷️ 類型

Agent Sandbox／Coding Agent Safety

---

## ✨ 一句介紹

讓 AI coding agent 可以在可拋棄環境裡測試，不污染本機。

---

## 👍 推薦原因

- 很適合鉗鉗未來測試工具或 repo。
- 可以降低未知套件、setup script、環境污染風險。
- 代表 Agent 落地時，Sandbox 會變成基本配備。

---

## 🧩 Cloudhaven 觀察

鉗鉗最怕的不是不會做，而是太乖：

```text
看到 README
↓
照著 setup
↓
裝了奇怪東西
↓
污染本機
↓
小查開始尖叫
```

TryCase 這類 disposable test environment 可以變成 Cloudhaven 的安全練習場。

---

## 🦞 龍蝦一句話

> 鉗鉗可以出去探險，但先穿防護衣，不要裸奔進 GitHub。
