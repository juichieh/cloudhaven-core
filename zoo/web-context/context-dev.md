# 🌐 Context.dev

> 讓 AI 更穩定抓取、豐富化與理解網頁內容的 Web Context API。

---

## 🏞️ Habitat

🟡 Research Habitat

## 🏷️ 類型

---

## ✨ 一句介紹

把網頁轉換成 Agent 可理解、可使用的任務上下文。

---

## 👍 推薦原因

- Agent 不只需要「打開網頁」，還需要穩定理解網頁內容。
- 很適合鉗鉗巡 GitHub / HN / Product Hunt 的底層零件。
- 代表 Browser Agent 從「點網頁」往「理解網頁」進化。

---

## 🧩 Cloudhaven 觀察

鉗鉗未來巡網路需要的不只是瀏覽器，而是：

```text
網頁內容
↓
整理成上下文
↓
判斷是否值得收藏
↓
輸出圖鑑候選
```

Context.dev 這類工具可以作為 Web Context Layer 的參考。
